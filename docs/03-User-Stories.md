# User Stories — MapSafe

## As a user, I want to:

### Account & Access
- Sign in using Google so I can access the platform easily

---

### Location Interaction
- View a map showing safety levels of different areas
- Search for a location on the map
- Zoom into specific areas to see details

---

### Ratings
- Rate an area based on my experience
- Select how large an area I want to rate (radius)
- Add optional comments explaining my experience

---

### Trust & Verification
- Only submit a rating if I am physically present in the location
- Only submit one rating per visit to avoid spam

---

### Information Viewing
- See average safety ratings for an area
- View recent user feedback
- See how safety has changed over time

---

## As an admin (future)

- Remove false or abusive ratings
- Monitor suspicious activity
