# MapSafe 

A community-driven safety map that helps people make informed travel decisions through verified location-based safety ratings.

---

##  Problem

People travelling or moving through unfamiliar areas often lack reliable, real-time information about how safe those places feel. Traditional map services provide navigation but not community-driven safety insights.

---

##  Solution

MapSafe allows users to:
- Rate the safety of locations they have physically visited
- Leave optional explanations or incident reports
- View a live safety map with colour-coded areas (green → red)
- See trends based on recent community feedback

---

##  Core Concept

- Green = Safe
- Yellow = Moderate
- Orange = Risky
- Red = Dangerous

Each area’s score is based on verified user visits and weighted recent activity.

---

##  Target Users

- Tourists
- E-hailing drivers
- Students
- Delivery drivers
- Residents in new cities

---

##  Tech Stack (Planned)

- Frontend: React + TypeScript + Vite
- Backend: Node.js + Express
- Database: PostgreSQL (PostGIS later)
- Maps: Google Maps API
- Auth: Google OAuth
